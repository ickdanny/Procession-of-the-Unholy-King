Controls

Menu Navigation
z = select
x = back
arrow keys = navigation

Gameplay
shift = focus
z = attack
x = bomb
arrow keys = movement
esc = pause
ctrl = skip dialogue

Left Handed Mode
fslash = shoot
period = bomb
wasd = movement
